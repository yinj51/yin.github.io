首页

Global counts

  • Connections：连接数
  • Channels：频道数
  • Exchanges：交换机数
  • Queues：队列数
  • Consumers：消费者数

交换机页面

队列页面

  • Name：消息队列的名称，这里是通过程序创建的
  • Features：消息队列的类型，durable:true为会持久化消息
  • Ready：准备好的消息
  • Unacked：未确认的消息
  • Total：全部消息
  • 备注：如果都为 0 则说明全部消息处理完成
